# dotfiles-redux
Better (hopefully speedier) dotfiles. 

## Things to install

Prerequisites:
- Homebrew: `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`
	- Git: `brew install git`

Useful things:
- Autojump
- NPM
- rbenv
- hub


## How to use

Run push script:

```bash
./pu.sh
```
