# dotfiles-redux
Better (hopefully speedier) dotfiles. 

## Things to install

Prerequisites:
- Homebrew: 

```shell
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
- Git: 

```shell
brew install git
```

Useful things:
- Autojump
- NPM
- rbenv
- hub


## How to use

Run push script:

```shell
./pu.sh
```
